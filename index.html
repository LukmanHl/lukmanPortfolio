About

Hi, I’m Lukman Hammed a curious, creative, and ever-evolving Fullstack Developer with a background in Computer Science.My journey into tech started with a love of creating things users can touch and see. I dove headfirst into the front end, mastering the skills to create clean, responsive websites using HTML, CSS, JavaScript, and frameworks like React and Tailwind CSS.
